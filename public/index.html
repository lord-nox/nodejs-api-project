API Documentatie

  • GET /users: Haalt alle gebruikers op
  • GET /users/:id: Haalt een specifieke gebruiker op
  • POST /users: Voegt een nieuwe gebruiker toe
  • Structure: 
                  {
                "name": "string",
                "email": "string",
                "age": "number",
                "phone": "+32 456 78 90 12" // Optioneel telefoonnummer in correct formaat
              }
  • PUT /users/:id: Update een bestaande gebruiker
  • DELETE /users/:id: Verwijdert een gebruiker
  • GET /newsPosts: Haalt alle nieuwsartikelen op
  • GET /newsPosts/:id: Haalt een specifiek nieuwsartikel op
  • POST /newsPosts: Voegt een nieuw artikel toe
  • Structure: 
                  {
                "title": "string",
                "content": "string",
                "author": "string",
                "startDate": "YYYY-MM-DD", // Optioneel startdatum in correct formaat
                "endDate": "YYYY-MM-DD"   // Optioneel einddatum in correct formaat
              }
  • PUT /newsPosts/:id: Update een bestaand artikel
  • DELETE /newsPosts/:id: Verwijdert een artikel


Paginatie

  • GET /users/paginated?limit={number}&offset={number}: Haalt een beperkt aantal gebruikers op, beginnend vanaf de opgegeven offset.
  • GET /newsPosts/paginated?limit={number}&offset={number}: Haalt een beperkt aantal nieuwsartikelen op, beginnend vanaf de opgegeven offset.

Parameters:

  • limit: Het maximale aantal resultaten dat moet worden opgehaald (standaard: 10).
  • offset: Het aantal resultaten dat moet worden overgeslagen (standaard: 0).

Voorbeeld:

        GET /users/paginated?limit=5&offset=10

Dit haalt 5 gebruikers op, beginnend bij de 11e gebruiker.



Zoeken

  • GET /users/search: Zoek gebruikers op basis van naam of e-mail
  • Query Parameters:
    • name: (optioneel) De naam van de gebruiker
    • email: (optioneel) Het e-mailadres van de gebruiker
  • Voorbeeld: 
                    GET /users/search?name=John
                GET /users/search?email=john@example.com


Geavanceerd zoeken

  • GET /users/advanced-search: Zoek gebruikers op basis van meerdere velden.
  • Query Parameters:
    • name: De naam van de gebruiker
    • email: Het e-mailadres van de gebruiker
    • phone: Het telefoonnummer van de gebruiker
  • Voorbeeld: 
                    GET /users/advanced-search?name=John&email=john@example.com
  • GET /newsPosts/advanced-search: Zoek nieuwsartikelen op basis van meerdere velden.
  • Query Parameters:
    • title: De titel van het artikel
    • author: De auteur van het artikel
    • startDate: Zoek vanaf deze datum (YYYY-MM-DD)
    • endDate: Zoek tot deze datum (YYYY-MM-DD)
  • Voorbeeld: 
                    GET /newsPosts/advanced-search?title=Breaking&author=John&startDate=2024-01-01
